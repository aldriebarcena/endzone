# Fantasee

Fantasy football live feed iOS app — live NFL game events re-displayed as fantasy scoring notifications, with league import and custom point settings.

- [DESIGN.md](DESIGN.md) — architecture, scope decisions, cost reality, build order
- [PROJECT_PLAN.md](PROJECT_PLAN.md) — task checklist tracking progress against the build order

## Layout

- `ios/` — Swift/SwiftUI client
- `backend/` — Python AWS Lambda backend (deployed via SAM)
